# Baltimore Armour 11G Aspire — Soccer Tracker

## How to deploy to Vercel (one time setup)

1. Go to vercel.com and sign in with Google
2. Click "Add New Project"
3. Click "Upload" (not GitHub)
4. Drag and drop the entire armour-aspire folder
5. Click Deploy
6. Done — Vercel gives you a URL like armour-aspire.vercel.app

## How to update the app

When I send you a new version:
1. Go to vercel.com
2. Click your project
3. Click "Deployments" → "Deploy" → upload the new folder
4. Your URL stays the same, data in Firebase is untouched

## Features
- Live game tracking with clock
- Firebase real-time save — never lose data
- Season stats, player stats, Optimum XI
- Works on any phone via the Vercel URL
